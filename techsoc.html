⚀ GIS: geographic information systems
• ESRI ArcGIS: beloved, powerful & proprietary
• QGIS: open source & scrappy
• misc other: special for web maps, design, etc.
⚁ data layers:
• GIS maps are composed of map layers, drawn in a particular order
• each layer is a dataset that represents a discrete entity (e.g. streets, trees, parks) a map layer defines how a GIS dataset is symbolized and labeled (that is, portrayed) in your map views NOTE: WE NEED A VISUAL HERE
⚂ data types: vector and raster
• a data type is a way of representing data
• all GIS data types consist of latitude and longitude (an x and y coordinate)
• the difference is what data is stored in each point/pixel
vector data:
• is split into three types: polygon, line and point data
• can store many attributes per point or pixel
raster data:
• is also called grid data, and it represents surfaces
• only stores one attribute per point or pixel, e.g. elevation or color ADD LINKS TO WORDS RASTER AND VECTOR TO MAKE THEM STAND OUT
• at the most basic level, think of vector data as a data table, and think of raster data as an image
• today, we are only dealing with vector data
• more info.

⚃ object types for vector data:
• points: a single pixel with a single x,y coordinate
• lines: two points or more connected with lines
• polygons: a bounded area of points connected by lines

⚄ projection systems
• a map is flat (2D). the world is not (?D).
• displaying a 3D object on a 2d surface requires some distortion
• there are many different ways of doing that distortion, and each has different virtues and vices http://www.radicalcartography.net/index.html?projectionref http://www.quadibloc.com/maps/mapint.htm http://geokov.com/education/map-projection.aspx Projections are also classified in terms of their properties. Specifically, on the basis of two very important properties: whether they are conformal, or equal-area, or neither. A conformal projection maintains the shape of small regions, so angles at any point are correct, although sizes will change. An equal-area projection, on the other hand, maintains size at the expense of shape. Maintaining both size and shape, of course, requires a globe.

⚀ navigate ArcMap + make a document (#1)
• create folder and give it a name with no spaces
• go to programs ➔ ArcGIS ➔ ArcMap (v 10.6.1)
• select "blank map"
• save to your folder: a doc in ArcMap is called an .mxd file make a folder on the desktop (or wherever) and give it a name (make sure there are no spaces in your folder name) we will be using ArcMap today, which is the 2D mapping software that is part of the ESRI ArcGIS suite (ESRI also makes 3d rendering software, called ArcScene and other fun things) first, go to programs -> ArcGIS -> ArcMap (should be version 10.6.1 but not a huge deal if not) select "blank map" (you may need to navigate through "New Maps" -> "My Templates" -> "Blank Map" in the left-hand table of contents) save your file - ArcMap files are saved as an .mxd (Map Exchange Document) which are very portable
⚀ navigate ArcMap + make a document (#2)
• the layer panel/TOC is your friend
• note the zoom, pan, and select buttons you can double-click on an .mxd file once it is saved to open it and ArcMap will launch data view is for working in, and layout view is for getting things print-ready the layer panel, or TOC, displays your layers, which you can turn on and off, drag to reorder, and right-click on to view their properties and attributes
